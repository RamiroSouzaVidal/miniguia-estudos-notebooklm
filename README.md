# 📖 Miniguia de Estudos

## O que é Engenharia de Prompts?

É a prática de elaborar instruções claras e específicas para que modelos de Inteligência Artificial produzam respostas mais precisas, completas e úteis.

---

## Boas práticas

* Definir um objetivo.
* Informar o contexto.
* Especificar o formato desejado.
* Informar o público-alvo.
* Solicitar exemplos.
* Pedir respostas organizadas.

---

## Técnicas

### Zero-shot

Nenhum exemplo é fornecido ao modelo.

Ideal para tarefas simples.

---

### One-shot

É fornecido apenas um exemplo.

Ajuda a IA a entender o formato esperado.

---

### Few-shot

São apresentados vários exemplos antes da solicitação principal.

Produz respostas mais consistentes.

---

### Chain of Thought

Solicita que o modelo explique o raciocínio passo a passo antes da resposta final.

---

### Role Prompting

Define um papel para o modelo.

Exemplo:

> "Você é um professor universitário."

# 📚 Glossário

| Conceito         | Definição                                      |
| ---------------- | ---------------------------------------------- |
| Prompt           | Instrução enviada ao modelo de IA              |
| LLM              | Grande Modelo de Linguagem                     |
| Token            | Unidade básica processada pela IA              |
| Contexto         | Informações adicionais fornecidas ao modelo    |
| Zero-shot        | Técnica sem exemplos                           |
| One-shot         | Técnica com um exemplo                         |
| Few-shot         | Técnica com poucos exemplos                    |
| Chain of Thought | Raciocínio passo a passo                       |
| Role Prompt      | Definição de um papel para o modelo            |
| Hallucination    | Resposta incorreta apresentada como verdadeira |

# 💡 Prompts Reutilizáveis

### Para estudar

```text
Explique este assunto como um professor universitário.
```

### Para revisar

```text
Resuma este conteúdo em tópicos.
```

### Para memorizar

```text
Crie um mapa mental em texto.
```

### Para praticar

```text
Crie cinco questões de múltipla escolha sobre este assunto.
```

### Para entrevistas

```text
Explique este conceito como se fosse uma pergunta de entrevista técnica.
```

### Para comparar conceitos

```text
Monte uma tabela comparando os principais conceitos.
```
